What is ASyncQuery.js?

ASyncQuery.js is a JavaScript Library that allows to make ASynchronous calls of functions. It allows to make several queries, repeating a function over and over or repeating a function a limited number of times. Basically, it allows asynchronous in JavaScript. It has been tested successfully in Chrome (& Chromium), Firefox & Internet Explorer, it should also work in Opera & Safari (I couldn't test it there, but if you did so, please write me an e-mail. )
